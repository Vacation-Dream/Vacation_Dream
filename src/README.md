주인공이름: 본인선택 (15살)
여자이름: 서아 (15살)

1. 여름방학을 맞이해 할머니댁에 놀러가게된 주인공은 옆집 손녀 서아(동네유명 미녀)를 만나게 되는데...
2. 서아랑 가까워지기 위해 주인공이 선택지를 선택하고 서아의 호감도를 쌓아 지역 최고 개꿀잼축제날 그녀에게 고백할 것이다.
3. 호감도에 따라 서아의 반응이 달라지고 주인공의 미래가 달라집니다.
